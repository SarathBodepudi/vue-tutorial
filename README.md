# VueJS 3 Tutorial 

## Clone Repository
1. Clone this repository
	```
	git clone https://github.com/SarathBodepudi/vue-tutorial.git
	cd vue-tutorial
	```
1. Change to folder you want to run example
	```
	cd 01-getting-started
	```

1. Install the npm packages

   ```
   npm install
   ```	
   
1. Run the app!
	```
	npm run serve
	```
